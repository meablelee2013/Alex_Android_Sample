解决启动问题时，有向无环图 拓扑排序

### 启动优化的本质，就是解决任务的依赖性问题，依赖问题的本质就是数据结构的问题

### 一个有向无环图(AVO网络)无法从某个顶点出发经过若干条边回到该点，那么这个图就是一个有向无环图，简称DAG图。DAG图常用用来表示事件之间的驱动依赖关系，管理任务之间的调度。拓扑排序是对一个有向图构造拓扑排序的过程
